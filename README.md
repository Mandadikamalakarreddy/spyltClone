## 🚀 Introduction

Welcome to the SPYLT website - a premium protein energy drink brand's digital experience that combines cutting-edge web technologies with award-winning design principles. This project showcases advanced GSAP animations, smooth scrolling experiences, and responsive design patterns that create an immersive user journey.

Built with Next.js 15 and the latest React 19, this website demonstrates modern web development practices while delivering exceptional performance and user experience.

## ⚙️ Tech Stack

- **Framework**: Next.js 15.4.1 with Turbopack
- **Frontend**: React 19.1.0 with TypeScript 5
- **Styling**: Tailwind CSS v4.1.11
- **Animations**: GSAP 3.13.0 with React integration
- **Icons**: React Icons 5.5.0
- **Responsive**: React Responsive 10.0.1
- **Development**: ESLint, PostCSS

## ✨ Features

This website includes:

- 🎭 **Advanced GSAP Animations**: Smooth parallax effects, scroll-triggered animations, and timeline controls
- 📱 **Fully Responsive Design**: Optimized for mobile, tablet, and desktop experiences
- ⚡ **Next.js 15 Performance**: Lightning-fast loading with Turbopack development server
- 🎨 **Modern UI/UX**: Award-worthy design with smooth transitions and micro-interactions
- 🧩 **Component Architecture**: Modular React components with TypeScript support
- 🎪 **Interactive Elements**: Flavor sliders, video pins, and dynamic content sections
- 🔄 **Smooth Scrolling**: ScrollSmoother integration for buttery-smooth navigation
- 🎯 **SEO Optimized**: Proper metadata and semantic HTML structure

## 🤸 Quick Start

Follow these steps to get the project running locally:

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd spyltclone
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the website.

## 🛠️ Available Scripts

- `npm run dev` - Start development server with Turbopack
- `npm run build` - Build the production application
- `npm run start` - Start the production server
- `npm run lint` - Run ESLint for code quality checks

## 📱 Responsive Design

The website is fully responsive and optimized for:

- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px  
- **Desktop**: 1024px+

All animations and interactions are touch-friendly and perform smoothly across devices.

## 🎞️ Animation Features

### GSAP Integration
- **ScrollTrigger**: Scroll-based animations and reveals
- **ScrollSmoother**: Buttery-smooth scrolling experience
- **Timeline Controls**: Complex animation sequences
- **Parallax Effects**: Depth and movement on scroll
- **Text Animations**: Character and word-based reveals
- **Clip-path Transitions**: Modern geometric transitions

### Interactive Elements
- Flavor selection sliders with dynamic backgrounds
- Video pinning and scroll-controlled playback
- Smooth navigation with active states
- Loading screens with brand animations

## 🔧 Development

### Key Components

- **HeroSection**: Landing area with brand messaging
- **FlavorSection**: Interactive flavor selection
- **NutritionSection**: Product benefits and information
- **TestimonialSection**: Customer reviews and social proof
- **FooterSection**: Contact and brand information

### Custom Hooks

- `useFontsLoaded`: Ensures animations wait for typography to load

### Constants

- Flavor configurations and product data
- Animation timing and easing presets
- Responsive breakpoint definitions
